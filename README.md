# CLI To-Do App

A simple command-line to-do list manager written in Node.js.

## Features
- Add tasks
- View tasks
- Delete tasks
- Persists tasks to `tasks.json`

## Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/CLI-Todo-app-Nodejs.git
   ```
2. Install dependencies (Node.js required):
   ```
   npm install
   ```
3. Run the script
   ```
   node app.js
